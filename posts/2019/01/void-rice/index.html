void-rice

Date [ Jan 30, 2019 ] Tags [ void-rice good-stuff foss linux suckless ]

Hello world.

I’ve decided to upload my suckless tools to github, so here’s void-rice. It can be barely called a desktop environment, but it features most things you need after a fresh install of Void Linux.

I’ve been using this setup for the last months and I’m really pleased by the native multihead (monitor) support with xrandr. void-rice features a custom patched version of the dwm window manager with system tray support and sane keybindings, dmenu, slstatus for the statusbar info and a patched st terminal emulator. It also includes multiple GTK themes and sane dependencies.

Installing

The automated install script is made for Void Linux but void-rice can be built on most linux distributions by installing the corresponding dependencies and manually building and installing each suckless tool.

NOTE: If the network info is not displayed correctly in the status bar you may need to replace wlan0 with your network interface in software/slstatus/config.h, rebuild slstatus and restart dwm.

The build.sh script just runs make clean && make in each one of the directories under “software/”, while install.sh just runs make install as root.

I will probably remove vpm from the next release to make the install.sh script portable, and I’m thinking about packaging this as a Void Linux metapackage.

Customizing

Simply edit config.h in each program’s directory in software/
